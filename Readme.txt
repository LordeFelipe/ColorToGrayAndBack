Tema:1 (Color to Gray and Back: Color Embedding into textured Gray Images)

Compilação
A compilação no MATLAB é feita simplesmente aperta RUN basta manter o arquivo das funções na mesma pasta dos programas principais

Execução
1-EmbeddingImage.m:
->Programa que cria as imagens texturizadas
->Para utilizá-lo, insira as imagens que deseja utilizar na pasta "IN" .As imagens texturizadas serão geradas na pasta "TEXTURE".
->Esse programa só aceita imagens em PNG

2-ColorRecovery.m
->Programa que recupera a cor das imagens, calcula o psnr e cria imagens com aumento de saturação
->Será perguntado ao usuário se ele deseja efetuar cada uma das operações descritas acima
  
OBS:
1-Apague as imagens na pasta RECOVERED e TEXTURE depois de criá-las para evitar problemas na próxima vez que rodar o código
2-As imagens dentro da pasta IN são as que foram utilizadas no relatório
